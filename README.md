# Excel-project-Coffee-Sales

# ☕ Coffee Sales Data Analysis Project

## 📌 Project Overview

This project analyzes coffee sales data using Microsoft Excel.  
The objective is to extract meaningful business insights such as total revenue, best-selling products, customer behavior, and sales trends using data cleaning, pivot tables, and dashboard visualization.

---

## 🎯 Objectives

- Analyze total sales revenue
- Identify best-selling coffee types
- Perform country-wise sales analysis
- Study customer purchase behavior
- Track monthly and yearly sales trends
- Create an interactive dashboard
